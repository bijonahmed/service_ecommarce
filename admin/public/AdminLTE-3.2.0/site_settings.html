Site status
Site domain name address

1. If the site domain name is not set, the current site domain name address will be used [ https://hello-test8.net/ ]

2. If the attachment and static address are not set, the static domain name address of the current site will be used [ https://hello-test8.net/public/ ]

3. If the server is not set with public as the root directory, then configure [attachment cdn domain name, css/js static file cdn domain name] here and add public later, such as: https://hello-test8.net/public/ public/

When running the project in command line mode, this area address must be configured, otherwise some addresses in the project will be missing domain name information.

Please do not configure it randomly. Wrong address will cause the website to be inaccessible (the address configuration starts with http). If your website is configured with https, it will start with https.
